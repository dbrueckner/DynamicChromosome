# DynamicChromosome	


DynamicChromosome contains code for analysing the coupled motion of two DNA loci on a chromosome, and for inferring paired and transcriptional states.

**Reference**: 
    David Brückner, Hongtao Chen, Lev Barinov, Benjamin Zoller and Thomas Gregor, 
    https://www.biorxiv.org/content/10.1101/2023.01.18.524527v1
